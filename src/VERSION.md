## 功能清单

### ✅ 核心功能

- [x] **饮食记录**
  - 快速记录（餐别选择）
  - 手动输入食物热量
  - 食物库（基础数据）
  - 快捷记录按钮

- [x] **运动记录**
  - 16 种预设运动
  - 自定义运动
  - 运动消耗计算（MET 公式）
  - 运动日志列表

- [x] **热量计算**
  - BMR 计算（3 种公式可选）
    - Mifflin-St Jeor（默认）
    - Harris-Benedict（健身人群）
    - Katch-McArdle（需要体脂率）
  - TDEE 计算
  - 每日预算计算
  - 目标缺口计算

- [x] **体重追踪**
  - 体重记录
  - 体重变化统计
  - 历史记录列表

- [x] **数据统计**
  - 今日概览（进度环）
  - 缺口趋势（周/月视图）
  - 三餐分布饼图
  - 达成率统计
  - 本周汇总

- [x] **设置**
  - 个人信息（性别/年龄/身高/体重）
  - 目标设置（目标体重/周期）
  - BMR 公式选择
  - 活动水平选择

- [x] **数据管理**
  - 导出 JSON 备份
  - 导入 JSON 恢复
  - 清除所有数据
